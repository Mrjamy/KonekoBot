# Koneko's Command list.

#### *Music, games, fun commands and much more!*

### Gambling

> /choice
>> arguments: [choices...]  
>> Choose from the given options split by ","  

### Games

> /roll  
>> arguments: [die]  
>> Rolls a die.  
  
> /rps  
>> arguments: <rock|paper|scissors>  
>> Play rock paper scissors.
